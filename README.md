# Soum_backend

## Getting Started

 <p> to run server in Development Mode<p>

```

npm install
npm run dev

```

 <p> to run server Test Mode<p>

```
npm run test

```

 <p> to run Graph Ql Api or run swager documentation you have to run server first<p>

```

npm run dev

```
